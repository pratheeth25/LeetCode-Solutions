class Solution {
    public int[] intersection(int[] nums1, int[] nums2) {

        Set<Integer> ans = new HashSet<>();
        Set<Integer> res = new HashSet<>();

        for (int a : nums1) {
            ans.add(a);
        }

        for (int a : nums2) {
            if (ans.contains(a)) {
                res.add(a);
            }
        }

        int[] p = new int[res.size()];
        int i = 0;

        for (int val : res) {
            p[i++] = val;
        }

        return p;
    }
}

We are not reprodcuing exact intersection instead we are just show casing elements present in both array not duplicates and can be in any order.
