class Solution {
    public int finalValueAfterOperations(String[] operations) {
        int ans = 0;
        for(String a:operations){
            if(a.equals("X++") || a.equals("++X")){
                ans++;
            }
            if(a.equals("X--") || a.equals("--X")){
                ans--;
            }
        }
        return ans;
    }
}

Here X++ and ++X are same 
SO only two operations are present.
